Это обычный JAVA PROJECT, используется для генерации объектов-сущностей которые сохраняются в БД.
В папке generatedDTO генерируются эти классы.
Затем эти классы необходимо копировать в Android project. Лучше всего всегда сравнивать что изменилось, а то можно перезатереть нужные методы, поля, импорты.
!!!!!!!!!!!!!!!!!!!!!!
DaoMaster.java после генерации нужно смерживать вручную. Там устанавливается версия БД и вставляются справочники.
